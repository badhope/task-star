## 如果出错

### 错误 1: ChromeDriver 问题

```
✗ 浏览器启动失败：Message: unknown error: cannot find Chrome binary
```

**解决**: 安装 Chrome 浏览器
- 下载：https://www.google.com/chrome/
- 安装后重新运行

### 错误 2: 网络问题

```
✗ 填写失败：Message: timeout
```

**解决**: 
- 检查网络连接
- 确保能访问 https://v.wjx.cn/vm/w7MiF02.aspx#

### 错误 3: 验证码

程序会暂停等待，需要手动完成验证码

## 中止任务

按 `Ctrl+C` 可以随时中止任务

## 需要修改什么？

### 修改填写次数

打开 `simple_fill.py`，找到最后一行：

```python
filler.run(times=100, interval=5)  # 改为需要的次数
```

### 修改间隔时间

```python
filler.run(times=100, interval=3)  # 改为 3 秒间隔
```

### 修改答案

找到 `_fill_question` 方法中的答案列表：

```python
answers = ["非常好", "比较满意", "还可以", "支持发展", "值得推广"]
```

改为需要的答案。
