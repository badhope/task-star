# 问卷执行说明

## 任务信息

**问卷链接**: https://v.wjx.cn/vm/w7MiF02.aspx#  
**问卷名称**: AI 驱动智能招聘与人才画像影响因素调查问卷  
**填写次数**: 100 次  
**预计耗时**: 15-25 分钟

## 配置详情

### 基本配置
```yaml
general:
  questionnaire_url: "https://v.wjx.cn/vm/w7MiF02.aspx#"
  fill_times: 100
  interval_seconds: 5
  headless: false
  auto_submit: true
  retry_count: 3
  page_timeout: 30
```

### 答题策略
```yaml
strategy:
  multi_choice:
    min_select: 2
    max_select: 3
  fill_blanks:
    9: ["AI 招聘提高了效率", "希望加强算法公平性", "需要完善相关法规", "建议加强人机协同", "技术还需进一步成熟"]
```

## 问卷结构分析

该问卷包含 9 道题目：

1. **性别** - 单选题（随机选择）
2. **年龄段** - 单选题（随机选择）
3. **行业** - 单选题（随机选择）
4. **职位** - 单选题（随机选择）
5. **工作年限** - 单选题（随机选择）
6. **企业规模** - 单选题（随机选择）
7. **是否引入 AI 招聘** - 单选题（随机选择）
8. **影响因素评价** - 多选题（14 个子题目，每个随机选择 2-3 项）
9. **其他建议** - 填空题（从答案池随机选择）

## 注意事项

1. **网络稳定性**: 确保网络连接稳定，避免页面加载超时
2. **浏览器版本**: 确保 Chrome 浏览器已安装且版本较新
3. **填写频率**: 建议不要设置过短的间隔时间（至少 3 秒），避免被封禁
4. **验证码处理**: 如遇到验证码，程序会暂停等待手动完成
5. **资源占用**: 运行时会占用一定的 CPU 和内存资源

## 常见问题

### 1. ChromeDriver 版本不匹配
```bash
pip install --upgrade webdriver-manager
```

### 2. 未找到 Chrome 浏览器
安装 Chrome：https://www.google.com/chrome/

### 3. 页面加载超时
- 检查网络连接
- 增加 `page_timeout` 配置
- 尝试关闭后台运行的其他程序

### 4. 提交失败
- 检查问卷链接是否正确
- 确认问卷未过期
- 尝试增加 `interval_seconds` 间隔时间
